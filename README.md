# 🧩 System Design Scenarios

> Scenario-based system design questions with detailed, interview-ready solutions.

![System Design](https://img.shields.io/badge/System%20Design-2C3E50?style=flat-square&logo=diagramsdotnet&logoColor=white)
![Distributed Systems](https://img.shields.io/badge/Distributed%20Systems-4479A1?style=flat-square)

A growing set of **real-world system design scenarios** — each worked through end to end: requirements, capacity estimates, data models, API design, trade-offs, and bottlenecks.

## 📂 What's inside

- [`Scenarios/`](Scenarios/) — one folder per scenario, each with the problem statement and a structured solution.

## 🎯 How to use it

1. Pick a scenario.
2. **Try designing it yourself first** — sketch the high-level design, data flow, and scaling strategy.
3. Compare with the solution and note the trade-offs you missed.

Great for interview prep and for sharpening real-world design intuition.
